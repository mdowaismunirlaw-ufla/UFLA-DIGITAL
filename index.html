⭐ Netlify Drop EASIEST
Drag and drop your HTML file — gets a public URL in 10 seconds. No signup required. Free forever for static sites.
1 Click "↓ Download JSON" above, then "↓ CSV" — save both files near your HTML file.
2 Go to netlify.com/drop
3 Drag your city_court_cases_v3.html file onto the page
4 You get a URL like random-name-123.netlify.app — share it!
→ netlify.com/drop
GitHub Pages FREE FOREVER
Free hosting with a permanent URL (yourname.github.io/court-cases). Best for long-term use.
1 Create free account at github.com
2 Create a new repository, name it court-cases
3 Upload your HTML file, rename it to index.html
4 Go to Settings → Pages → Source: main branch → Save
5 Your site is live at yourname.github.io/court-cases
→ github.com

☁ Connect Google Drive
Google Drive requires a free Google Cloud API key. This is a one-time 5-minute setup. You do this once and never again.
Step by step setup:
1
Go to console.cloud.google.com and sign in with your Google account
2
Click Create Project → name it Court Cases → Create
3
Go to APIs & Services → Enable APIs → search for Google Drive API → Enable
4
Go to Credentials → Create Credentials → OAuth 2.0 Client ID → choose Web Application → Add your Netlify URL under Authorized Origins → Copy the Client ID
5
Also create an API Key → restrict it to Drive API → Copy the API Key
6
Paste both below — then click Save & Reconnect
